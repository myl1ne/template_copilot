# Sample Project - Example Usage

This directory contains an example of how the Template Copilot template looks after customization.

## Overview

This sample demonstrates a fictional "TaskFlow" project - a task management application. It shows how all placeholders have been replaced with actual project information.

## What's Included

- `README.md` - Customized main README
- `docs/project-overview.md` - Filled-in project overview
- `docs/roadmap.md` - Sample roadmap with real milestones
- `docs/backlog.md` - Example backlog with tasks

## How This Was Created

```bash
# 1. Used the template_copilot template
# 2. Ran the quick start script
python quick_start.py

# Input provided:
# - Project name: TaskFlow
# - Description: A modern task management application
# - Repository: https://github.com/example/taskflow
# - Version: 0.1.0
# - License: MIT
# - Author: TaskFlow Team

# 3. Further customized the documentation
# 4. Validated completeness
python validate_template.py
```
