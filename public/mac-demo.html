🎨 MAC Framework

MeshAsCode - A recursive framework for creating complex 3D meshes using only code

Key Features

  • Code as Data: Mesh definitions stored as editable code strings
  • Recursive Composition: MACs can contain other MACs
  • Save/Load/Edit: Full persistence with JSON and JavaScript export
  • Visual Editor: Interactive UI for creating and editing MACs
  • Template Library: Pre-built components ready to use
  • Transform System: Easy positioning, rotation, and scaling

Quick Example

const tree = new MAC('group') .add('cylinder', { radiusTop: 0.2, radiusBottom: 0.3, height: 2, material: { color: 0x4a2511 } }) .position(0, 1, 0) .add('cone', { radius: 1, height: 2, material: { color: 0x2d5016 } }) .position(0, 3, 0); const mesh = tree.build(); scene.add(mesh);

Use Cases

• Procedural Generation: Create variations programmatically
• Asset Libraries: Build reusable component collections
• Level Design: Compose complex scenes from simple parts
• Rapid Prototyping: Quickly iterate on 3D designs
• Educational: Learn 3D modeling through code

Integration

The MAC framework integrates seamlessly with existing Three.js code and can be used alongside the traditional MeshLibrary system.
