# Ant Colony Simulator - Project Overview

## What is Ant Colony Simulator?

Ant Colony Simulator is an interactive 3D simulation that demonstrates emergent intelligence through ant colony behavior. Built with three.js, it visualizes how simple individual rules create complex, efficient colony-level behavior.

## Target Audience

This project is designed for:
- **Educators and Students** - Learning about emergent behavior, swarm intelligence, and ant colony optimization algorithms
- **Game Developers** - Exploring AI behavior patterns and pathfinding techniques
- **3D Graphics Enthusiasts** - Studying three.js implementation and 3D visualization
- **Anyone Curious** - People who want to watch mesmerizing ant behavior in a beautiful 3D environment

## Key Features

- **Intelligent Ant Behavior**: Ants autonomously explore, find food, and optimize paths using pheromone trails
- **Real-time 3D Visualization**: Smooth, interactive 3D environment powered by three.js with WebGL rendering
- **Pheromone Trail System**: Visual golden trails that ants leave and follow, fading over time
- **Dynamic Environment**: Food sources deplete as ants collect, creating realistic resource management
- **Interactive Controls**: Pause/play, reset, add ants, and freely navigate the 3D camera

## Quick Start

### Prerequisites
- Modern web browser with WebGL support (Chrome, Firefox, Safari, Edge)
- Node.js v14+ (for local development server)

### Installation
```bash
git clone https://github.com/myl1ne/template_copilot.git
cd template_copilot
npm install
```

### Basic Usage
```bash
npm start
```

The simulator will open automatically in your browser. Use your mouse to interact:
- **Left-click + drag** to rotate the camera
- **Right-click + drag** to pan
- **Scroll** to zoom in/out
