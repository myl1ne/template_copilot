# Ecosystem Sandbox

> A game running locally, which simulates an environment with lifeforms made of geometric primitives. Design the game around the evolution of the player's species, in a Spore's like fashion.

An interactive 3D evolution simulation game built with Three.js and React, where players guide geometric lifeforms through evolutionary stages in a dynamic ecosystem.

## 🚀 Quick Start

1. **Clone the repository** to your local machine
2. **Install dependencies** with `npm install`
3. **Start the development server** with `npm start`
4. **Open your browser** to http://localhost:3000 to begin evolving!

## 🎮 Game Concept

**Ecosystem Sandbox** is a 3D evolution simulation where:

1. **Geometric Lifeforms** - Creatures built from primitive shapes (spheres, cubes, cylinders)
2. **Environmental Adaptation** - Species evolve based on ecosystem pressures and player choices
3. **Spore-like Progression** - Guide your species through multiple evolutionary stages
4. **Local Simulation** - Runs entirely in your browser with no external dependencies

## 🛠️ Technology Stack

- **Frontend**: React 18+ with hooks and context
- **3D Graphics**: Three.js for WebGL rendering
- **Build Tool**: Create React App or Vite
- **Styling**: CSS Modules or Styled Components
- **State Management**: React Context + useReducer
- **Physics**: Custom lightweight physics engine

## 🌱 Evolution Mechanics

The game features multiple interconnected systems:

1. **Species Design**: Customize creature appearance and abilities
2. **Environmental Pressure**: Dynamic ecosystem challenges
3. **Genetic Traits**: Heritable characteristics affecting survival
4. **Population Dynamics**: Realistic birth/death cycles
5. **Resource Competition**: Limited food and territory
