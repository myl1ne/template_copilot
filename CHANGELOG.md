# Changelog

All notable changes to the Template Copilot project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-10-28

### Added
- **Interactive quick-start script** (`quick_start.py`) for automated template customization
- **Template validation script** (`validate_template.py`) to check for unreplaced placeholders
- **GitHub Actions workflow** for automated documentation validation
- **Issue templates** for bugs, features, and documentation updates
- **CONTRIBUTING.md** with contribution guidelines
- **LICENSE** file (MIT License)
- **template-config.example.yml** showing advanced configuration options
- **.gitignore** file for common development artifacts
- **Enhanced README** with better structure, badges, and usage examples

### Changed
- Improved documentation structure with clearer sections
- Updated README to version 1.1.0
- Enhanced project metadata and organization

### Fixed
- N/A (initial feature release)

## How to Use This Changelog

When contributing to Template Copilot:

1. Add your changes under the appropriate section (Added, Changed, Deprecated, Removed, Fixed, Security)
2. Use the "Unreleased" section for changes not yet in a release
3. Update the version number and date when creating a release
4. Follow [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) format

## Categories

- **Added**: New features
- **Changed**: Changes in existing functionality
- **Deprecated**: Soon-to-be removed features
- **Removed**: Removed features
- **Fixed**: Bug fixes
