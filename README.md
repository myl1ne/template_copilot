# RimWorld Tower Defense Mod

A comprehensive tower defense modification for RimWorld featuring autonomous towers, manual wave triggering, and research-gated upgrades.

## About This Mod

This mod transforms RimWorld into a tower defense experience by adding:
- **Autonomous Towers**: Self-operating defensive structures that target hostiles
- **Wave System**: Manual enemy wave triggering via Wave Console
- **Upgrade System**: Tower upgrades through the Bills/Recipes system
- **Research Tree**: Progressive unlocking of towers and upgrades

For detailed information about building, installing, and using this mod, see **[RimWorld_Mod_README.md](RimWorld_Mod_README.md)**.

## Quick Start

1. Set the `RIMWORLD_DIR` environment variable to your RimWorld installation
2. Run `./build.sh` (Linux/Mac) or `build.bat` (Windows)
3. Install the mod to your RimWorld Mods folder
4. Enable in RimWorld's mod manager
