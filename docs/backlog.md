## 🏷️ Issue Labels Guide

When creating new GitHub issues, use these labels to automatically sync with this backlog:

- `priority: high` - High priority items (foundation and core features)
- `priority: medium` - Medium priority items (gameplay features)
- `priority: low` - Low priority items (polish and optimization)
- `type: bug` - Bug fixes
- `type: feature` - New features
- `type: enhancement` - Improvements to existing features
- `type: documentation` - Documentation updates
- `type: setup` - Project setup and configuration tasks

### Additional Project-Specific Labels
- `component: 3d-rendering` - Three.js and WebGL related tasks
- `component: evolution` - Evolution system and genetics
- `component: ui` - React components and user interface
- `component: physics` - Movement and collision systems
- `stage: alpha` - Alpha development phase tasks
